Touchy.js

JavaScript library for touch optimized input elements

Wheel selector

Show me!
Selected: 17.0
Touchy.wheel($('#wheelTouchy'), {
	data: {"17.0": "17,0°", "21.0": "21,0°", "21.5": "21,5°", "22.0": "22,0°",
		"22.5": "22,5°", "23.0": "23,0°", "28.0": "28,0°"},
	selection: function(value) {
		$('#wheelSelected').html(value);
	},
	value: function() {
		return $('#wheelSelected').html();
	}
});

More to come

Touchy.js is in a very early state and currently planned are the following data inputs:

  • Wheel selection for multiple entries
  • Time selection
  • Date selection
  • Data selection with longer labels

Technology

Touchy.js is pure JavaScript powered SVG and CSS3!

Used libraries

  • jQuery
  • snap.svg
